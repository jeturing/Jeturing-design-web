# argo-web — Catálogo de Paletas

10 paletas curadas para evitar diseños genéricos de IA.  
Usar siempre **variables CSS** — nunca valores hardcoded.

---

## Regla de uso

```css
/* ✅ Correcto */
.card { background-color: var(--bg-surface); }

/* ❌ Incorrecto */
.card { background-color: #111827; }
```

## Ejemplo de uso universal

```css
/* 1. Importar la paleta elegida */
@import url('./palettes/01-dark-tech.css');

/* 2. Usar SIEMPRE variables — nunca valores hardcoded */
.card {
  background-color: var(--bg-surface);
  color: var(--text-primary);
  border-left: 3px solid var(--color-primary);
}

.metric-highlight {
  color: var(--color-accent);
  font-weight: 700;
}

.btn-primary {
  background-color: var(--color-primary);
  color: var(--text-primary);
}

.btn-primary:hover {
  background-color: var(--color-accent);
  color: var(--bg-base);
}
```

## En Tailwind CSS

```typescript
// tailwind.config.ts — extender con los tokens
export default {
  theme: {
    extend: {
      colors: {
        primary: 'var(--color-primary)',
        accent:  'var(--color-accent)',
        base:    'var(--bg-base)',
        surface: 'var(--bg-surface)',
      }
    }
  }
}
```
